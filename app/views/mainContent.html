Using ngPluralize

Partially compatible with native ngPluralize directive.

$scope.pluralization = {
    numCount: 1
};
<div class="form-group">
    <input type="number" min="0" class="form-control" data-ng-model="pluralization.numCount" data-i18n-attr="{placeholder: 'demo.numberPrompt'}">
    <div class="alert alert-warning">
        <span class="glyphicon glyphicon-question-sign"></span>
        <span data-ng-pluralize data-count="pluralization.numCount" data-when="{ '0': '{{ pluralization.options.zero | i18n }}', '1': '{{ pluralization.options.one | i18n }}', 'other': '{{ pluralization.options.many | i18n }}' }" ></span>
    </div>
</div>

Limitations

Full compatibility cannot be reached due to limitations of the ng-framework. Right now it is impossible to place the data-count attribute anywhere in the resultant string. There exists a known workaround using $filter('i18n') or locale.getString() directly in your controller or else you could expose a getPlurals() function to the ngPluralize directive. The newly created function should internally address the value of count.

This is somewhat wieldy and cumbersome as it requires you to track offsets manually, the same can also be achieved thru the i18n directive albeit with one extra degree of complexity; the caveat being that of interpolation along with $watchers and/or ng-bind

In such situations it is good to note that locale.ready() is not mandatory as angular's page $digest cycle will fire periodically and take care of it for you.

Also, strings may not be passed to the i18n filter(s) of the ngPluralize directive as is, in the markup.

$scope.pluralization = {
    numCount: 1
    options: {
        zero: 'plurals.zeroPluralized',
        one: 'plurals.onePluralized',
        many: 'plurals.manyPluralized'
    },
    getPlural: function () {
        var n = $scope.pluralization.numCount ,
            s;
        switch (n) {
        case null:
        case undefined:
        case 0:
            s = $scope.pluralization.options.zero;
            break;
        case 1:
            s = 'plurals.onePluralizedNum::' + n;
            break;
        default:
            s = 'plurals.manyPluralizedNum::' + n;
            break;
        }
        return s;
    },
    getPlurals: function () {
        var n = $scope.pluralization.numCount ,
            s;
        switch (n) {
        case null:
        case undefined:
            s = '';
            break;
        case 0:
            s = $filter('i18n')($scope.pluralization.options.zero);
            break;
        case 1:
            s = $filter('i18n')('plurals.onePluralizedNum', n);
            break;
        default:
            s = $filter('i18n')('plurals.ManyPluralizedNum', n);
            break;
        }
        return s;
    },
    getWithOffs: function () {
        var n = $scope.pluralization.numCount ,
            s;
        switch (n) {
        case 0:
            s = $scope.pluralization.options.zero;
            break;
        case 1:
            s = 'plurals.onePluralizedOfs';
            break;
        case 2:
            s = locale.getString('plurals.twoPluralizedOfs', $scope.user.name);
            break;
        case 3:
            s = locale.getString('plurals.threePluralizedOfs', $scope.user.name);
            break;
        default:
            s = locale.getString('plurals.morePluralizedOfs', [$scope.user.name, (n - 2)]);
            break;
        }
        return s;
    }
};
<div class="form-group">
    <input type="number" min="0" class="form-control" data-ng-model="pluralization.numCount" data-i18n-attr="{placeholder: 'demo.numberPrompt'}">
    <div class="alert alert-warning">
        <span class="glyphicon glyphicon-warning-sign"></span>
        <span data-i18n="{{ pluralization.getPlural() }}"</span>
    </div>
    <div class="alert alert-warning">
        <span class="glyphicon glyphicon-warning-sign"></span>
        <span data-ng-bind="pluralization.getPlurals()"</span>
    </div>
    <div class="alert alert-warning">
        <span class="glyphicon glyphicon-question-sign"></span>
        <ng-pluralize data-count="pluralization.numCount" data-offset="2" data-when="{ '0': '{{ pluralization.getWithOffs() | i18n }}', '1': '{{ pluralization.getWithOffs() | i18n }}', '2': '{{ pluralization.getWithOffs() }}', 'one': '{{ pluralization.getWithOffs() }}', 'other': '{{ pluralization.getWithOffs() }}' }" ></ng-pluralize>
    </div>
</div>
